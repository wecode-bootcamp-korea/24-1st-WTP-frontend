- 디자인이 깔끔하고 컨텐츠 관련 API 활용이 원활하여 배울 점이 많을 것 같아서 선정되었습니다.

## 적용 기술 및 구현 기능

<br>

### 적용 기술

<br>

> -Front-End : javascript, React.js framwork, styled-components<br>
> -Back-End : Python, Django web framework, MySQL, Bcrypt, pyjwt, AWS, S3<br>
